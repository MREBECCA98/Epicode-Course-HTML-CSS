CORSO WEB DEVELOPER

Questa è una landing page promozionale per il corso Web Developer di Epicode School.
Il progetto mostra le informazioni principali sul corso, i benefici e le opportunità offerte, utilizzando un layout semplice e moderno.

Tecnologie utilizzate:
-HTML5 per la struttura del sito
-CSS3 per lo styling e il layout
-FontAwesome per le icone

Funzionalità principali:
-Sezioni divise per contenuti: “Tutti ti cercheranno”, “Epicode garantisce”, “Carte informativo”
-Layout responsive (adattabile a diversi dispositivi)
-Footer con informazioni di contatto

Obiettivo del progetto:
Esercitarsi nella creazione di una landing page con HTML e CSS, applicando concetti di struttura, layout e design responsive.
